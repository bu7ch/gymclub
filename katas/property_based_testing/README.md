Property-Based Kata
===================

A simple exercise to demonstrate property-based testing, in a browser
context.

Dependencies
------------

nodejs and npm

Install
-------

```
npm install
```

be sure of using a recent version of node... for example with nvm
```
nvm use v0.10.30
```

Run it
------

```
npm run test-auto
```
